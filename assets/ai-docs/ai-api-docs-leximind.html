API Documentation: LexiMind Predictive Diagnosis

By Leslie Amadi

Overview

LexiMind provides predictive diagnosis for chronic illnesses using patient data. Developers can integrate the model via REST API.

Authentication

Use your API key in the Authorization header.

Base URL

https://api.leximind.ai/v1

Endpoints

POST /predict

Description: Predict the likelihood of chronic illnesses for a patient.

Request Example (JSON):

{
  "patient_data": {
    "age": 55,
    "blood_pressure": 135,
    "cholesterol": 200,
    "blood_sugar": 110
  }
}

Response Example:

{
  "prediction": {
    "diabetes": 0.78,
    "heart_disease": 0.45,
    "kidney_disease": 0.12
  },
  "recommendation": "Consult doctor for diabetes monitoring."
}

Python Example:

import requests

url = "https://api.leximind.ai/v1/predict"
headers = {"Authorization": "Bearer YOUR_API_KEY"}
data = {
  "patient_data": {
    "age": 55,
    "blood_pressure": 135,
    "cholesterol": 200,
    "blood_sugar": 110
  }
}

response = requests.post(url, headers=headers, json=data)
print(response.json())

GET /status

Description: Check if the model is online.

{
  "status": "online",
  "uptime": "99.9%"
}

Error Codes

  • 401: Invalid API key
  • 400: Bad request
  • 500: Server error
