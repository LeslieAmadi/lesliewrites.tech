# Leslie Writes — Portfolio Launch Checklist

## ■ A. Visual Design Check
• Branding & Theme — Ensure colour palette and fonts are consistent.
• Layout — Verify page structure and spacing.
• Images — All images display correctly with alt text.
• Mobile View — Test responsive layout on phones and tablets.
## ■■ B. Functional Check
• Navigation — All internal links (Home, About, Portfolio, Blog, Contact) work.
• Buttons — Test hover states and click actions.
• Blog — Ensure all posts display and RSS feed is valid.
• PDFs — Check all downloads open correctly.
## ■ C. Technical Check
• File Structure — Confirm files follow standard hierarchy.
• Metadata — Add title, description, and viewport tags.
• Performance — Compress images and test with PageSpeed.
• Accessibility — Use proper headings and text contrast.
## ■ D. Deployment Check
• GitHub Pages — Site loads from public repo.
• DNS — Add CNAME for lesliewrites.tech.
• Final Validation — Run broken link and HTML
