Risk Scoring API

Authoritative system contract for predictive risk evaluation services.

← Back to Portfolio
On this page: Endpoint · Request Schema · Response Schema · Error Model · System Guarantees

Endpoint

POST /risk-score

Accepts structured input data and returns a normalised predictive risk score.

Request Schema

{
  "age": number,
  "bmi": number,
  "bp": number
}

All fields are required. All values must conform to validated numeric ranges.

Response Schema

{
  "risk_score": number,
  "confidence": number
}

risk_score — Normalised prediction value (0–1)
confidence — Model confidence level (0–1)

Error Model

{
  "error_code": string,
  "message": string,
  "trace_id": string
}

Errors are deterministic, traceable, and auditable.

System Guarantees

  • Schema stability
  • Versioned evolution
  • Backward compatibility
  • Deterministic response structure
  • Auditability of outputs

This API functions as a system contract between platform services and consuming applications.
