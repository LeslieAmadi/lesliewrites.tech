3. Regression Testing Template

This regression suite ensures that every new model release maintains or improves quality without introducing regressions.

3.1 Scope

  • Intent classification accuracy
  • Multilingual fluency (EN/DE)
  • Tone consistency
  • Hallucination reduction
  • Latency and cost

3.2 Test Categories

Intent Classification Tests

Test IDDescriptionExpected Output
IC-01Recognise billing queriesCorrect intent
IC-02Technical troubleshootingCorrect intent
IC-03Ambiguous queriesHighest-probability valid intent

Multilingual Robustness

IDLanguageScenarioExpectation
MLR-01GermanFormal inquiryCorrect and fluent
MLR-02GermanColloquialReasonable interpretation
MLR-03EnglishEmotional toneSoft, empathetic reply

Hallucination Tests

IDCaseExpected Behaviour
HL-01Unsupported productSafe fallback
HL-02Incorrect factsAsk for clarification
HL-03Unsafe contentBlock + safe response

3.3 Pass/Fail Criteria

  • No accuracy drop > 2%
  • No new hallucinations introduced
  • Latency must not increase by >10%
  • Safety tests must pass 100%
