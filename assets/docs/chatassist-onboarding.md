# Onboarding Guide: ChatAssist AI

## 1. Introduction

ChatAssist AI is an intelligent support assistant designed to help customer service teams respond faster, maintain consistent tone, and reduce workload. This guide walks you through setup, configuration, and first use.

---

## 2. Requirements

- Organisation admin account  
- Supported browser: Chrome, Edge, Firefox  
- API key with “write” permissions  
- Access to ChatAssist dashboard  

---

## 3. Setup Steps

### Step 1 — Log in
Visit: https://dashboard.lexitech.ai/chatassist  
Use your organisation credentials.

### Step 2 — Connect your help centre  
You can connect:
- Zendesk  
- Intercom  
- Freshdesk  

ChatAssist imports:
- Macros  
- Articles  
- Categories  

### Step 3 — Configure Tone & Style  
Set defaults:
- Professional  
- Friendly  
- Neutral  
- Technical  

### Step 4 — Add Team Members  
Invite agents → Assign roles:
- Admin  
- Editor  
- Agent  

---

## 4. How ChatAssist Works

### 4.1 Draft Suggestions  
When an agent selects a ticket, ChatAssist:
1. Reads the user question  
2. Searches knowledge base  
3. Generates a suggested response  

### 4.2 Rewrite Options  
- Shorten  
- Expand  
- Formalise  
- Simplify  

### 4.3 Safety Filters  
Prevents:
- Harmful content  
- Sensitive data leakage  
- Unsupported claims  

---

## 5. Best Practices

- Always review suggestions  
- Add examples to improve model results  
- Maintain updated knowledge base  
- Use rewrite options for consistency  

---

## 6. Troubleshooting

| Issue | Cause | Solution |
|-------|--------|----------|
| No suggestions | Missing API key | Re-add key |
| Slow responses | Heavy load | Try refresh |
| Incorrect tone | Tone config wrong | Update style settings |
