# Neural Networks

## Overview
A **Neural Network** is a type of machine learning model inspired by the human brain.  
It consists of layers of interconnected nodes, or *neurons*, that learn to recognise complex patterns in data.  

Neural Networks form the foundation of modern **Deep Learning**, powering everything from voice recognition to image classification.
